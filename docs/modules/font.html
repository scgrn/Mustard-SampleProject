Module font

Font loading and rendering.

Currently only supports AngelCode BM font format with XML descriptor

Functions

AB.font.loadFont (filename, index)
Loads a font

Parameters:

  • filename Filename.
        Pass in "default1" for a built-in 8x16 font
        Pass in "default2" for a built-in 8x8 font
        Pass in "default3" for a built-in 3x5 font
  • index (optional) Font index

Returns:

    font index
AB.font.printString (layer, index, x, y, scale, alignment, str)
Prints a string

Parameters:

  • layer Rendering layer
  • index Font index
  • x X position
  • y Y position (baseline)
  • scale Scale
  • alignment Alignment (AB.font.LEFT, AB.font.CENTER, AB.font.RIGHT)
  • str String to print
AB.font.stringLength (index, str, scale)
Calculates width of string

Parameters:

  • index Font index
  • str String
  • scale Scale

Returns:

    Width of string in pixels
AB.font.setColor (index, r, g, b, opacity, additivity)
Sets font (premultiplied) color

Parameters:

  • index Font index
  • r (1.0) Red color component
  • g (1.0) Green color component
  • b (1.0) Blue color component
  • opacity (1.0) Opacity
  • additivity (0.0) Additivity
