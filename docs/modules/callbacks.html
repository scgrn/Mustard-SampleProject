Functions

AB.loadConfig ()
Load configuration. Called before window and renderer are created. This should set a global table: videoConfig = { title, xRes, yRes, fullscreen, vsync }
AB.init ()
Called once before the main loop is entered. All subsystems are guaranteed to be stood up.
AB.update ()
Updates logic. Called 60 times a second.

Renders graphics
