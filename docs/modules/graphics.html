AB.graphics.resetVideo ()
Sets the video mode. Only necessary upon changes to video mode. Reads a global table: videoConfig = { title, xRes, yRes, fullscreen, vsync }
AB.graphics.getDesktopResolution ()
Gets desktop resolution

Returns:

  1. xRes X resolution
  2. yRes Y resolution
AB.graphics.clear (r, g, b, a)
Clears the screen or current canvas

Parameters:

  • r (0.0) Red color component
  • g (0.0) Green color component
  • b (0.0) Blue color component
  • a (1.0) Alpha component
AB.graphics.loadSprite (filename, collisionMask, index)
Loads a sprite

Parameters:

  • filename Filename
  • collisionMask (true) If a collision mask should be created
  • index (optional) Sprite index

Returns:

    sprite handle
AB.graphics.loadAtlas (filename, width, height, collisionMask, index)
Loads a series of sprites from a spritesheet

Parameters:

  • filename Filename
  • width Width of each sprite
  • height Height of each sprite
  • collisionMask (true) If a collision mask should be created
  • index (optional) First sprite index

Returns:

  1. first sprite handle
  2. number of sprites loaded
AB.graphics.addToAtlas (index)
Adds a sprite into the current atlas queue. This should be done at startup before any rendering occurs. Sprites that are commonly used together can be batched into a single atlas to improve performance by minimizing texture switches.

Parameters:

  • index Sprite index
AB.graphics.buildAtlas ()
Builds a sprite atlas. Call this after several calls to AB.graphics.addToAtlas
AB.graphics.renderSprite (layer, index, x, y, z, angle, scaleX, scaleY)
Queues a sprite to be renderer on the screen or current canvas.

Parameters:

  • layer Rendering layer. A default layer of 0 is provided
  • index Sprite index
  • x X position
  • y Y position
  • z (-1) Z position
  • angle (0) Rotation
  • scaleX (1) Scale X
  • scaleY (scaleX) Scale Y

AB.graphics.createLayer (index, depthSorting)
Creates a new rendering layer. (RenderLayer internally). Layers are renderered back to front, largest indices first. A default layer of 0 is provided

Parameters:

  • index Layer index
  • depthSorting (false) Whether this layer needs to support depth sorting

Returns:

    layer index

Removes a rendering layer

  • index Layer index
AB.graphics.addColorTransform (index, transform, r, g, b, a)
Adds a color transform for layer. Chained with any previous color transforms

Parameters:

  • index Layer index
  • transform Color transform
  • r Red (will also be interpreted as theta or value)
  • g Green
  • b Blue
  • a Alpha

See also:

    colorTransforms

Usage:

  • AB.graphics.addColorTransform(0, AB.graphics.colorTransforms.MULTIPLY, 0.75, 0.25, 0.5)
  • AB.graphics.addColorTransform(0, AB.graphics.colorTransforms.SATURATE, 0.5)
